    Quick Start (Development)¶

    Prerequisites¶

    • Node.js 20+ — nodejs.org
    • Git — git-scm.com
    • Neon account — neon.tech (free tier works)
    • Monnify Sandbox — sandbox.monnify.com

    1. Clone & Install¶

    git clone git@github.com:remitpro/schoolpay.git
cd schoolpay
npm install --legacy-peer-deps
cp .env.example .env
# → Edit .env with your credentials

    2. Database Setup¶

    npm run db:push      # Sync schema to Neon
npm run db:rls       # Apply Row Level Security
npm run db:triggers  # Setup DB triggers
npm run db:seed      # Load demo data (Greenwood International)

    3. Start Dev Servers¶

    # Terminal 1: Express backend
npm run server:dev   # Starts on http://localhost:5001

# Terminal 2: Vite frontend
npm run dev          # Starts on http://localhost:8080

    4. Verify¶

    Open http://localhost:8080 → click "Sign in as Demo Admin" → you should see the Greenwood dashboard.

    # Quick health check
curl http://localhost:5001/health
# → {"status":"ok","timestamp":"..."}

    Roles & Permissions¶

    Role Database Value Access Scope
    Super Admin super_admin Platform-wide; bypasses RLS; sees all schools
    School Owner owner Created during onboarding; full school control
    Admin admin Equivalent to owner for operations
    Bursar bursar Finance & store only; no academic/HR access
    Teacher teacher Academics only; their assigned class
    Parent (no login) Portal access via Student ID + phone number

    Command Description
    npm run dev Start Vite frontend dev server (port 8080)
    npm run server:dev Start Express backend with hot reload (port 5001)
    npm run build Production frontend build
    npm run lint ESLint check
    npm run test Vitest unit tests
    npm run test:e2e Playwright E2E tests
    npm run db:push Push Drizzle schema to database
    npm run db:generate Generate migration SQL files
    npm run db:studio Open Drizzle Studio visual DB browser
    npm run db:seed Seed demo data
    npm run db:rls Apply Row Level Security policies
    npm run db:triggers Setup database triggers
    npm run db:init Production database initialization
    npm run docs:serve Live docs preview at http://127.0.0.1:8000
    npm run docs:build Build static docs site to /site/
